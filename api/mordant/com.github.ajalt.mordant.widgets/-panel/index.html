A box drawn around another widget, with optional top and bottom titles.

Example

terminal.print(Panel(
  "my panel content",
  title = "title",
  bottomTitle = "subtitle",
  titleAlign = TextAlign.LEFT,
  bottomTitleAlign = TextAlign.RIGHT
))
